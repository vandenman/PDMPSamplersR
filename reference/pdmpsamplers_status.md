# Show the status of the PDMPSamplers.jl installation

Displays the current version of PDMPSamplers.jl and whether it is
installed from a local path or from GitHub.

## Usage

``` r
pdmpsamplers_status()
```

## Value

Invisible `NULL`. Called for its side effects.

## See also

[`pdmpsamplers_setup`](https://vandenman.github.io/PDMPSamplersR/reference/pdmpsamplers_setup.md)
to perform the initial setup,
[`pdmpsamplers_update`](https://vandenman.github.io/PDMPSamplersR/reference/pdmpsamplers_update.md)
to update Julia dependencies.
