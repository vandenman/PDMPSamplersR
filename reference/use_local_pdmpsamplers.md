# Use a local version of PDMPSamplers.jl

Switch the Julia project to use a local (development) copy of
PDMPSamplers.jl instead of the GitHub version, or switch back to the
GitHub version.

## Usage

``` r
use_local_pdmpsamplers(path = NULL)
```

## Arguments

- path:

  Character string giving the path to the local PDMPSamplers.jl
  directory. If `NULL` (default), switches back to the GitHub version.

## Value

Invisible `NULL`. Called for its side effects.
